# snr
String Number Recognition project

[JavaDoc](https://kindratsm.github.io/snr/javadoc/)

## Test the project
Test the project, run the command:
```
mvn test
```

## Build the project
Clean and build the project, run the command:
```
mvn install
```

## Run the project
Run the command:
```
mvn exec:java
```
or
```
java -jar snr-0.0.1.jar
```
